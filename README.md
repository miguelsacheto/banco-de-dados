# 🗄️ Banco de Dados

Repositório dedicado aos projetos, atividades e materiais desenvolvidos na disciplina de **Banco de Dados**. Aqui estão reunidos os trabalhos práticos relacionados à modelagem, criação e manipulação de bancos de dados.

## 🎯 Sobre a disciplina

A matéria de Banco de Dados tem como objetivo desenvolver competências relacionadas a:

- Conceitos fundamentais de bancos de dados e SGBDs (Sistemas de Gerenciamento de Banco de Dados)
- Modelagem de dados (Modelo Entidade-Relacionamento)
- Linguagem SQL (DDL, DML, DCL)
- Criação, consulta e manipulação de tabelas
- Normalização de dados
- Relacionamentos entre tabelas (um-para-um, um-para-muitos, muitos-para-muitos)

## 🏅 Certificação

Este repositório inclui um certificado de conclusão relacionado aos estudos de Banco de Dados, comprovando o aprendizado dos conceitos e práticas abordados na disciplina.

## 🛠️ Conceitos e tecnologias abordados

- Banco de dados relacionais
- SQL (Structured Query Language)
- Modelagem e design de banco de dados
- Chaves primárias e estrangeiras
- Integridade e consistência de dados
